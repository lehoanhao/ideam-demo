# Task 01: 顧客 (Customers) ページ再構築

## 概要
提案ページ（proposals）のパターンに基づいて顧客ページを再構築する。
サイドバーナビ付きレイアウト + リスト/詳細ページのパターン。

## ファイル構成

### ページ
- `app/pages/customers.vue` → proposals.vueパターンのサイドバー付きレイアウト
  - サイドメニュー: 顧客一覧、タグで絞り込み
  - 新規作成ボタン
- `app/pages/customers/[[tag]].vue` → 顧客リスト（タグフィルタ付き）
- `app/pages/customers/d/[[id]].vue` → 顧客詳細/作成/編集ページ（2カラムレイアウト）
  - 左: 基本情報フォーム（名前、ふりがな、連絡先、タグ）
  - 右サイドバー: タブ切り替え
    - 基本情報（読み取り用）
    - 対応履歴タイムライン（提案実績 → /proposals/d/xxx、営業活動 → /activities/xxx へリンク）

### 変更
- 既存の `app/pages/customers/index.vue` → `app/pages/customers/[[tag]].vue` に移動
- 既存の `app/pages/customers/[id].vue` → `app/pages/customers/d/[[id]].vue` に移動 + 2カラムレイアウトに変更

## 対応履歴タイムライン
- 提案実績リンク → `/proposals/d/{proposalId}`
- 営業活動リンク → `/activities/{activityId}`
- 時系列表示（新しい順）
