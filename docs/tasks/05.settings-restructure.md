# Task 05: 設定 (Settings) ページ再構築

## 概要
設定ページをサイドバーナビ付きに変更。タグ管理とアカウント管理をサブページとして追加。

## ファイル構成

### ページ
- `app/pages/settings.vue` → サイドバー付きレイアウト
  - サイドメニュー: タグ管理、アカウント管理
- `app/pages/settings/tags.vue` → タグ管理（作成・編集・削除）
- `app/pages/settings/accounts.vue` → アカウント管理（作成・編集・削除 + ロール・権限）
  - ロール: 営業、受け入れ担当、提案承認者（社長）など

### 変更
- 既存のsettingsページ構造をサイドバー付きに変更
- タグ管理を`/tags`から`/settings/tags`に移動
- アカウント管理は新規作成
