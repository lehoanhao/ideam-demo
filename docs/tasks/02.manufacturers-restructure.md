# Task 02: メーカー (Manufacturers) ページ再構築

## 概要
提案ページパターンに基づいてメーカーページを再構築する。

## ファイル構成

### ページ
- `app/pages/manufacturers.vue` → サイドバー付きレイアウト
  - サイドメニュー: メーカー一覧、カテゴリで絞り込み
  - 新規作成ボタン
- `app/pages/manufacturers/[[tag]].vue` → メーカーリスト
- `app/pages/manufacturers/d/[[id]].vue` → メーカー詳細/作成/編集ページ（2カラムレイアウト）
  - 左: 基本情報フォーム（名前、ふりがな、連絡先、商品カテゴリ）
  - 右サイドバー: タブ切り替え
    - 基本情報
    - 対応履歴タイムライン（過去の見積実績 → /proposals/d/xxx へリンク）

### 変更
- 既存の `app/pages/manufacturers/index.vue` → `app/pages/manufacturers/[[tag]].vue` に移動
- 既存の `app/pages/manufacturers/[id].vue` → `app/pages/manufacturers/d/[[id]].vue` に移動 + 2カラムレイアウトに変更

## 対応履歴タイムライン
- 過去の見積実績 → `/proposals/d/{proposalId}` へリンク
