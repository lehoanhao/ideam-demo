# Task 03: 営業活動 (Activities) ページ再構築

## 概要
Backlogライクな画面で営業活動を管理する。サイドバーナビ付き。

## ファイル構成

### ページ
- `app/pages/activities.vue` → サイドバー付きレイアウト
  - サイドメニュー: すべて活動、ステータス別フィルタ、日報
  - 新規作成ボタン
- `app/pages/activities/[[tag]].vue` → 活動リスト（ステータスフィルタ付き）
- `app/pages/activities/d/[[id]].vue` → 活動詳細/作成/編集（2カラムレイアウト）
  - 左: フォーム（タイトル、顧客名、種類、ステータス、活動日、担当者、窓口、意欲レベル、案件紐付け）
  - 右サイドバー: タブ切り替え
    - 基本情報
    - コメント・タイムライン
- `app/pages/activities/report.vue` → 日報ページ
  - 顧客単位/営業担当単位で切り替え
  - 日付選択

### 変更
- 既存の `app/pages/activities/index.vue` → `app/pages/activities/[[tag]].vue` に移動
- 既存の `app/pages/activities/[id].vue` → `app/pages/activities/d/[[id]].vue` に移動 + 2カラムレイアウトに変更

## 日報機能
- グループ: 顧客単位 or 営業担当単位
- 日付ピッカー
- 各エントリに活動種類、タイトル、説明を表示
